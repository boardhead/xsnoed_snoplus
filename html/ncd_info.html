NCD Info Window

The NCD Info window shows the NCD data of the current event in text form. All fields in the NCD data record are printed. The NCDTubeMap.dat file is used to determine the NCD string name in this printout. If an entry doesn't exist in NCDTubeMap.dat for a tube, then the string name will show up as "?" or "unknown", and the NCD tube will not be displayed in the 3-D view or the NCD Map window.
