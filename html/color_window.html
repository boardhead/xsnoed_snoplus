Colors Window

The Colors window is used to edit the XSNOED image colors. This window is accessed through the File/Settings/Colors menu item.

To change a color, first make a selection from the list of available colors at the bottom of the window, then adjust the Intensity slider and click in the color wheel to change the selected color. Alternatively, the Red/Green/Blue components of the new color can be entered directly in the text fields to the right. The color component values should be in the range 0 to 255.

Press OK to apply all changes and close the Colors window, or use Apply to implement the changes and leave the window open.

The Revert button reverts the colors of the current color set to the values they had when the Colors window was opened.

The Cancel button closes the Colors window. The image colors remain at their current settings.

The current image color settings are saved with the rest of the XSNOED settings when Save Settings is selected from the File menu.

Note: The Greyscale colors are derived from the luminance of the corresponding colors in the regular color set. To change the level of a Greyscale color, change the intensity of the associated color in the regular color scheme. The exceptions are the scale colors which are mapped into a linear grey scale and can't currently be modified.
