NCD Map Window

The NCD Map window shows an X-Y display of the NCD tube locations. The tube colors represent the value of the currently displayed NCD data type. The colors are determined by the X scale of the NCD Histogram, and the displayed data type is set using the Data menu.

The X-Y positions, string names and electronics channels are obtained from the NCDTubeMap.dat file. This file must exist in the XSNOED directory and be current to properly display the NCD hits in this window.

NCD tubes which are not hit, or which are cut from being displayed based on the hit mask set in the Display menu show up as empty white or grey circles respectively in this window.

The NCD tube map for this window is loaded dynamically depending on the run number of the data being viewed, so the tube positions may not appear exactly as pictured. Specifically, two spare counters (S1 and S2) and a random pulser (RP) may appear when viewing data for some runs. These counters do not exist physically in the detector, but they appear in this map for convenience.
