Display Menu

The Display menu provides control over various aspects of the display. A description of all Display menu items is given below.

Hits Submenu

The hits submenu is used to specify the types of PMT and NCD hits to display. A PMT will be displayed only if ALL of its corresponding attributes are selected in this menu. (ie. A normal PMT with an overscale value will be displayed only if both Normal and Overscale are selected in the menu.) The logic is different for NCD's however, where an NCD will be displayed if ANY of its attributes are selected.
Hits/All
Turns on display of basic PMT (Normal, Owl, Low Gain, Neck, BUTTS and FECD) and NCD (NCD Shaper, NCD MUX and NCD Scope) hit types.
Hits/None
Turns off display of all basic PMT and NCD hit types.
Hits/PMT Only
Turns on display of PMT hits, and turns off display of NCD hits.
Hits/NCD Only
Turns on display of NCD hits, and turns off display of PMT hits.
Hits/Normal
Toggles on and off display of normal PMT's. When
Hits/Owl
Toggles on and off display of outward looking PMT's. Outward looking PMT's are drawn in the location of their corresponding vertex on the PSUP for the purposes of display.
Hits/Low Gain
Toggles on and off display of low gain PMT's.
Hits/Neck
Toggles on and off display of neck PMT's (PMT's located in the neck of the acrylic vessel). Neck PMTs are displayed in the 3-D image and the crate view, but are not shown in other projections.
Hits/BUTTS
Toggles on and off display of BUTTS (Berkeley Underwater Tube Testing Sled) tubes. BUTTS PMTs are displayed in the 3-D image and the crate view, but are not shown in other projections.
Hits/FECD
Toggles on and off display of hits which occurred on channels with no PMT connected (Front End Card Digitizer channels). FECD hits are only displayed in the crate view.
Hits/NCD Shaper
Toggles on and off display of NCD hits where an NCD shaper fired. When on, causes NCD's to be displayed if the corresponding shaper fired, regardless of other attributes.
Hits/NCD MUX
Toggles on and off display of NCD hits where an NCD MUX fired. When on, causes NCD's to be displayed if the corresponding MUX fired, regardless of other attributes.
Hits/NCD Scope
Toggles on and off display of NCD hits where NCD scope data was available. When on, causes NCD's to be displayed if scope data exists, regardless of other attributes.
Hits/Discarded
Toggles on and off display of discarded PMT hits. Hits may be marked as discarded by using the center mouse button in any image window.
Hits/Underscale
Sets the display of PMT hits that are below the minimum histogram x scale.
Hits/Overscale
Sets the display of PMT hits that are above the maximum of the histogram scale.
Vessel
Toggles on and off display of acrylic vessel in the 3-D image window.
Hit Lines
Toggles on and off display of hit lines in the 3-D image window. Hit lines are drawn from the current fit vertex to each PMT hit. The color of each line corresponds to the current hit color scale. If no fit is available, hit lines are drawn from (0,0,0).
Fit Lines
Toggles on and off display of lines drawn from the fit vertex along the Cherenkov cone. Has no effect if a fit is not available.
Water Level(s) - plural for ROOT_FILE version only
Toggles the light water level display. Also toggles display of heavy water level for the ROOT_FILE version. The light water level is shown by a line drawn where it intersects the PSUP. The water level is taken from the water_level setting of the XSnoed resource file for non-ROOT versions, and from the water level database for the ROOT_FILE version. The water level is implemented in the display by adding a special fit of type "Water" for light water, and "D2O" for heavy water. As such, any fit display options also apply to the water level display. Note that both water levels only appear in the 3-D display (of the ROOT_FILE version), the other projections show the light water level but not the heavy water level.
3D Geodesic
Displays the 3-D SNO PSUP support frame view in the main image window.
3D Polar
Displays a 3-D polar coordinate view in the main image window.
3D No Frame
Displays a 3-D view with no detector wire frame in the main image window.
Flat Map
Displays the unfolded, flat PSUP view in the main image window.
Crate Map
Displays the crate map in the main image window.
Projection
Displays the map projection in the main image window. When this item is selected, and new Projection menu appears in the main menu bar. The Projection menu is described in more detail in the section on the Projection window.
NCD Map
Displays the NCD Map image in the main window.
White Backgnd
Selects the color set with a white background (the alternate colors) for all image windows.
Greyscale
Selects the greyscale color scheme for all image windows.
