NCD Histogram Window

The NCD Histogram window displays a histogram of the current selected NCD data type. The X scale of this histogram sets the color of the displayed NCD hits.

The NCD data type is selected through the Data menu, and can be set to display the shaper value, MUX hit count, shaper hit count, or scope hit count for each NCD tube. In the image on the right, "NCD MUX Hits" has been selected in the Data menu.

See the description of the PMT Histogram window for more information about using an XSNOED histogram.
