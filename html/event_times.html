Event Times Window

The Event Times window is a histogram of the 50 MHz clock value for events in the uncut history buffer. The currently viewed event is shown in red if it exists in the buffer. Times are given in seconds with respect to the zeroth entry of the history buffer (the most recent event read).

Note that orphan events will not be displayed in this window because they have no associated trigger time stamp.

The scales for the window can be changed by clicking or dragging with the mouse like the Histogram window. Once the scales have been changed, they will no longer automatically scale as new events come in. A quick click in histogram plot area restores the automatic scaling.
