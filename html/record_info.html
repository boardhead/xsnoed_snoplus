Record Info Window

The Record Info window gives information about header records in the data. This information is obtained from records received from the dispatcher or read from a ZDAB file.

This window can display information about the following types of records:

  • Run Record (RHDR)
  • Trigger Record (TRIG)
  • ECA Record (EPED)
  • Calibration Source (CAST)
  • AV Position (CAAC)
  • Laser Status (SOSL)
The displayed record type is selected from the menu at the top of the Record Info window. Note that because of the way XSNOED reads ahead of the currently displayed event, the information in this window may lead the displayed events. This is a particular problem if you want to analyze the sequence of EPED records. Perhaps a future version of XSNOED will cache these records and display them in the proper sequence with respect to the events.

Here is a description of the displayed fields for the Run Record type:

Run Number is the number of the current run.

Start Time is the SHaRC start time for the run.

Run Type gives a list of the conditions for the run (see table below for a complete list of definitions).

Sources gives a list of sources deployed for the run.

Crate Mask lists the crates which were masked into the global trigger for the run.
