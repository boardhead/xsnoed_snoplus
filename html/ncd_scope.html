NCD Scope Window

The NCD Scope window shows digitized NCD scope waveforms for all NCD scope channels. Note that most NCD events will only fire a single scope channel, however in sum mode the latest data for each scope channel is displayed.

The scales for all scope channels are tied together, so if you change one they all change. The traces for scope 0 and scope 1 are displayed in the same plot, and are differentiated by color. The colors set via the Colors settings, and are shared with the Ch1/Ch2 colors of the Trigger Scope.

The Display menu in this window allows you to set the scope channels to be displayed. By default all scope channels are displayed, but this can be changed to display any individual channel, or only channels with scope data. Note that if Channels with Data is selected, the displayed channels will only change if data is available (so zero channels will not be displayed).

The Data menu (ROOT_FILE version only) allows you to select display of Raw or AntiLog data. Antilogged data is displayed only if AntiLog is selected and the logamp calibration constants are available. When antilogged data is displayed, the label changes to "AntiLog 0" or "AntiLog 1" instead of "Scope 0" or "Scope 1" for each successfully antilogged trace. Note that the antilogged data values are multiplied by 1000 to bring them into roughly the same range as the raw values.
